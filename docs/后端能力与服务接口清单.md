# 后端能力与服务接口清单

本文档只描述**当前**后端形态：Python 网关通过 [app/gateway_api.py](/C:/Projects/githubs/aiagents-stock/app/gateway_api.py) 暴露页面快照和动作接口，复用 `app/` 下既有业务逻辑、SQLite、行情数据源和调度器。

## 1. 当前后端形态

- 网关入口：[app/gateway.py](/C:/Projects/githubs/aiagents-stock/app/gateway.py)
- 网关定义：[app/gateway_api.py](/C:/Projects/githubs/aiagents-stock/app/gateway_api.py)
- 默认服务地址：
  - 前端页面：`http://127.0.0.1:8501/main`
  - 健康检查：`http://127.0.0.1:8501/api/health`
- 单服务同端口提供：
  - SPA 静态页面
  - `/api/v1/*` 页面快照
  - `/api/v1/*/actions/*` 页面动作

当前主调用链：

`前端页面 -> gateway_api 快照/动作 -> 服务对象 -> SQLite / 数据源 / 调度器`

### 9.4 自动执行规则

当前保持严格规则：

- A 股必须满足一手 `100` 股
- 自动执行打开后才会自动成交
- 不满足条件时会保留跳过原因

常见跳过提示：
- `缺少有效最新价`
- `建议仓位不足买入一手`
- `当前无可卖持仓`
- `当前无可卖数量`

## 13. 当前后端边界

后续继续重构时，建议保持这些边界稳定：

1. 网关只负责快照与动作接口，不重写业务服务
2. 业务逻辑继续留在 `app/` 下现有服务模块
3. “我的关注”与量化候选池继续是两层对象
4. 量化模拟与历史回放继续共用同一量化候选池
5. 发现股票 / 研究情报只负责产出结果并手工推进到“我的关注”
