  - [ui/src/lib/api.ts](/C:/Projects/githubs/aiagents-stock/ui/src/lib/api.ts)
- 当前后端网关：
  - [app/gateway_api.py](/C:/Projects/githubs/aiagents-stock/app/gateway_api.py)
  - [app/gateway.py](/C:/Projects/githubs/aiagents-stock/app/gateway.py)

前端通过统一模式工作：

1. 页面进入某个路由，例如 `/main`
2. 调用对应快照接口，例如 `/api/v1/workbench`
3. 用户点击按钮后调用动作接口，例如 `/api/v1/workbench/actions/add-watchlist`
4. 动作完成后重新拉取当前页面快照

## 13. 当前页面能力的共同约束

后续页面继续重构时，应保持这些共性不变：

1. 所有页面通过 [app/gateway_api.py](/C:/Projects/githubs/aiagents-stock/app/gateway_api.py) 提供快照和动作接口
2. 前端只负责路由、展示和触发动作，不重写 Python 业务逻辑
3. `我的关注` 与 `量化候选池` 是两层不同对象
4. `量化模拟` 与 `历史回放` 共用同一量化候选池
5. 有股票输出的发现/研究页才能加入“我的关注”
